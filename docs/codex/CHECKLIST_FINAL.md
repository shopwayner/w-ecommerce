# Checklist Final de Entrega

Antes de finalizar qualquer tarefa, conferir:

- [ ] A tarefa foi cumprida conforme o objetivo.
- [ ] Nada fora do escopo foi alterado.
- [ ] Funcionalidades existentes foram preservadas.
- [ ] `lint` foi executado quando disponivel.
- [ ] `test` foi executado quando disponivel.
- [ ] `build` foi executado quando disponivel.
- [ ] Responsividade foi conferida quando houve alteracao visual.
- [ ] Erros no console foram conferidos quando houve alteracao visual ou interativa.
- [ ] Variaveis de ambiente foram preservadas e nenhum segredo foi exposto.
- [ ] Arquivos alterados foram listados.
- [ ] Pendencias, riscos ou ressalvas foram informados.
